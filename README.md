# Sistema de gestión de tareas - To-DO App

Aplicación web desarrollada con **HTML5, CSS3 y JavaScript**, diseñada para gestionar tareas de manera rápida, simple y eficiente.
El sistema permite crear, completar y eliminar tareas, guardando los datos en el navegador mediante **LocalStorage**.

## Funcionalidades

- Añadir tareas
- Marcar como completadas
- Eliminar tareas
- Guardado automático en localStorage
- Diseño responsive
- Interfaz limpia, moderna y profesional
